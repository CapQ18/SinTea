# 正弦茶 - 奶茶分享平台技术栈说明

## 一、项目概述

**项目名称**：正弦茶（SinTea）
**项目类型**：前端单页应用（SPA）
**项目定位**：奶茶爱好者社交分享平台

---

## 二、技术栈详情

### 2.1 前端框架

| 技术 | 版本 | 用途 | 说明 |
|------|------|------|------|
| React | 19.x | UI框架 | 组件化开发，虚拟DOM渲染 |
| TypeScript | 5.8.x | 类型系统 | 类型安全，智能提示 |

### 2.2 构建工具

| 技术 | 版本 | 用途 | 说明 |
|------|------|------|------|
| Vite | 6.3.x | 构建工具 | 极速热更新，Rollup打包 |

### 2.3 样式框架

| 技术 | 版本 | 用途 | 说明 |
|------|------|------|------|
| Tailwind CSS | 3.4.x | CSS框架 | 原子化CSS，响应式设计 |
| PostCSS | 10.5.x | CSS预处理 | 自动前缀，代码优化 |

### 2.4 路由与图表

| 技术 | 版本 | 用途 | 说明 |
|------|------|------|------|
| React Router DOM | 7.15.x | 路由管理 | 单页应用路由控制 |
| Recharts | 3.8.x | 图表库 | 雷达图、数据可视化 |

## 三、项目结构

```
├── src/
│   ├── components/          # UI组件
│   │   ├── FeedCard.tsx     # 动态卡片组件
│   │   ├── IPCharacter.tsx  # IP角色组件
│   │   ├── MilkTeaSprite.tsx # 奶茶精灵组件
│   │   ├── BottomNav.tsx    # 底部导航
│   │   └── NavBar.tsx       # 顶部导航
│   ├── pages/              # 页面组件
│   │   ├── Home/           # 首页
│   │   ├── Discover/       # 发现页（奶茶店展示）
│   │   ├── FeedHome/       # 动态首页
│   │   ├── FeedDetail/     # 动态详情页
│   │   ├── FeedPost/       # 发布动态页
│   │   ├── Profile/        # 个人主页
│   │   ├── Wishlist/       # 心愿单页
│   │   ├── Login/          # 登录页
│   │   └── Register/       # 注册页
│   ├── mock/               # Mock数据
│   │   └── index.ts        # 模拟数据（用户、评论、心愿单）
│   ├── services/           # 服务层
│   │   ├── authService.ts  # 认证服务
│   │   ├── mockData.ts     # 数据服务
│   │   └── postService.ts  # 帖子服务
│   ├── types/              # 类型定义
│   │   ├── index.ts        # 全局类型
│   │   ├── feed.ts         # 动态相关类型
│   │   └── user.ts         # 用户相关类型
│   ├── App.tsx             # 应用入口组件
│   ├── main.tsx            # React挂载入口
│   └── style.css           # 全局样式
├── public/                 # 静态资源
│   └── images/             # 图片资源
├── index.html              # HTML入口
├── package.json            # 依赖配置
├── vite.config.ts          # Vite配置
├── tsconfig.json           # TypeScript配置
├── tailwind.config.js      # Tailwind配置
└── postcss.config.js       # PostCSS配置
```

## 五、运行说明

```bash
npm install
```

```bash
npm run dev
```

### 5.3 生产构建

```bash
npm run build
```

### 5.4 预览构建结果

```bash

```

## 六、技术亮点

1. **组件化架构**：React组件化开发，代码复用性高
2. **类型安全**：TypeScript提供完整的类型支持
3. **响应式设计**：Tailwind CSS实现移动端适配
4. **虚拟列表**：优化大量数据渲染性能
5. **状态管理**：React Hooks管理组件状态
6. **代码规范**：统一的代码风格和命名规范
